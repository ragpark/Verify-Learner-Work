Moodle File Transfer Tool

This service must be launched from your Learning Management System via LTI 1.3.

  1. Register this app as an LTI tool in your LMS using {{ base_url }} as the launch URL.
  2. Launch the tool as an administrator to supply OAuth client details for your Moodle platform.
  3. Authorize with Moodle when prompted and use the interface to queue course file transfers to Azure Blob Storage.

Use the /healthz endpoint for a basic deployment check.
